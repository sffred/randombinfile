这个嘛，就是用来生成不可压缩的，毫无雷同的垃圾文件，用来填充空间的啦

源码见rp.c，可自行用参数g++ rp.c -o rp -static编译校验，安全无毒没有危险。

用法

rp有4个参数，-o,-n,-s,-d。-o 后跟输出路径，不能带空格，最后需要有斜杠或反斜杠，缺省输出到程序所在目录；-n后跟输出文件数量，缺省为4；-s后跟输出文件大小（以4KiB为单位），缺省为2^18（1GiB）；如果加上-d参数，则输出文件大小会有抖动，否则输出精确的指定大小。

编译好的rp.exe已在windows 1909 通过测试；rp已在Linux 4.4.0-18362-Microsoft通过测试。
